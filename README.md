# VoiceChatApp

Real-time voice dialogue app with LLM & TTS support.

## Features

- Real-time voice interaction
- LLM: OpenAI (GPT-4o) / Volcano (Doubao)
- TTS: ElevenLabs / Volcano TTS
- Material 3 design

## Getting Started

1. Go to Actions tab
2. Wait for build to complete
3. Download the APK

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
